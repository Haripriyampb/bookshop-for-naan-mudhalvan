# Bookshop Project

## Description
A Node.js project to manage books in a shop, including viewing books, searching by ISBN, author, title, and managing user reviews.

## Tasks
1. Get all books
2. Get book by ISBN
3. Get all books by author
4. Get all books by title
5. Get a book review
6. Register new user
7. Login as a user
8. Add/modify a review
9. Delete a review
10. Get all books using async/await
11. Search book by ISBN using async/callback
12. Search by Author using async/callback
13. Search by Title using async/callback
14. Submission via GitHub link

## Instructions
1. Run the server: `node shop.js`  
2. Use Postman to test the endpoints (GET, POST, PUT, DELETE).  
3. Screenshots of tasks are included in the repository.
